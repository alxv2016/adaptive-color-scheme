Adaptive color scheme with HSL (Hue, Saturation, Lightness)

HSL stands for hue, saturation and lightness. Unlike HEX values and RGB, HSL is much more intuitive and lets us have complete control over saturation and the lightness of the colour.

Using CSS variables and calc() and HSL, we can create an adaptive colour scheme that responds to user preference. We can easily define all the colours for light, dark and dim colour schemes by calculating the saturation and lightness of one brand colour.
