CSS & HSL Magic

Using CSS variables and HSL, we can create an adaptive color scheme that responds to a users preference. We can easily define all the colors for light, dark, and even a dim color scheme by dynamically controling the saturation and lightness of the brand color through CSS variables.

This demo of an adaptive color scheme built for Angular was created based on Adam Argyle's article: Building a color scheme. You can fine-tune this color scheme by playing with the hue in the settings and seeing it adapt to various theme preferences.
