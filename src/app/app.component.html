CSS & HSL Magic

Using CSS variables and HSL, we can create an adaptive colour scheme that responds to a users preference. We can easily define all the colours for light, dark, and even a dim colour scheme by dynamically controling the saturation and lightness of the brand colour through CSS variables.

This demo of an adaptive colour scheme built for Angular was created based on Adam Argyle's article: Building a color scheme. You can fine tune this color scheme by playing with the hue in the settings and see it adapt to various theme preferences.
